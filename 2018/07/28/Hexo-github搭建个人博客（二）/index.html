Hexo+github搭建个人博客（二）

发表于 2018-07-28 | 分类于 博客搭建

Next主题

###更换next主题，分成两步：
先下载next主题文件，再修改主题配置文件_config.yml中相关参数

  • 进入./blog/themes目录下，在git bash中输入：

    git clone https://github.com/iissnan/hexo-theme-next themes/next
    从这个github上下载next主题文件

  • 将themes/next中的next文件拿出来放在和landscape同级目录下比较方便查看。

  • 启用next主题
    克隆/下载完成后，打开themes下主题配置文件_config.yml,找到theme字段，将其改为next主题（冒号之后加空格）

    theme主题字段
    theme: next

  • 验证主题
    清除并重新生成hexo静态文件，启动本地服务器，然后通过http://localhost:4000预览博客页面

    hexo clean #清除静态文件
    hexo g #更新生成的静态文件
    hexo s #启动服务器

可以通过域名访问查看到刚刚更新的next主题设置。
