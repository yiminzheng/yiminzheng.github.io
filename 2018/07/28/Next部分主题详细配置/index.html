Next部分主题详细配置

发表于 2018-07-28 | 分类于 博客搭建

修饰Next主题博客页面

在Hexo中blog根目录下的_config.yml配置，包含Hexo本身的配置，称为站点配置文件，另一份 _config.yml位于blog/themes/next/下，可以自己配置页面外观，称为主题配置文件。

  • 基本配置信息

    博客标题、作者、描述、语言等等。


  • 菜单设置

    菜单包括：首页、关于、标签、分类、归档等。
    默认一开始只有首页和归档，我们可以自己添加菜单，打开主题配置文件找到menu Settings

打开注释就会增加菜单选项，自己选择。

  • 添加分类模块
    上面只是在页面上添加了相应菜单，点击的话会显示不可到达，那是因为你没有添加菜单对应的模块。

    1. 新建一个分类页面
      $hexo new page categories
    2. 执行完之后，source文件夹下就会出现categories/index.md,打开index.md,将title设置为title: 分类
    3. 打开主题配置文件找到menu，将categories取消注释即可。
    4. 将文章归入分类只需在文章顶部标题下添加categories字段，即可自动创建分类名并加入对应的分类中。
    5. 效果图
